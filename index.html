2.着物はいつ出現（しゅつげん）ですか？

1.江戸（えど）時代
2. 平安（へいあん）時代
3. 奈良時代
4. 室町（むろまち）時代

3.中国に、着物について、『格』の概念（がいねん）があって、何意味ですか？

1.着物のもよう(花纹的意思)
2. 着物の値段
3. 着物の正式（せいしき）レベル（等级）
4. 着物の色

4.通常（つうじょう）に以下の着物どちらが一番高いですか？

1.本振袖（ほんふりそで）
2. 付け下げ（つけさげ）
3. 浴衣（ゆかた）
4. 紬（つむぎ）

4. 打掛（うちかけ）
